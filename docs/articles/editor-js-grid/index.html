Use Editor.js more flexibly

Editor.js is one of block editors. The article data format is JSON not HTML, so it can manage article data more cleanly and formally.

Editor.js can’t nest blocks to keep simplicity. So it is useful for general articles that blogs, news and so on. But difficult for special layout that landing page and so on.

Therefore, I published a grid layout plugin for Editor.js.

Try it out!

editor-js-grid - npm

Concepts

editor-js-grid is a reduced plugin that provides only grid design system. editor-js-grid provides a minimum DOM that can edit plain text by default. By providing DOM in yourself, it can provide image, video and other contents.
